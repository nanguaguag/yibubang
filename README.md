## Features

### 💻 跨平台

你是否在烦恼电脑上没法刷题？来试试医不帮吧！支持：

- [x] 安卓端（universal、arm64、x86-84）
- [x] MacOS
- [x] Windows
- [x] Linux
- [ ] IOS/iPadOS（即将推出）
- [ ] ~~网页端（目前来说不可行）~~

### ✅ 本地化

没有网就刷不了题了？不存在的！

- 离线数据库
- 本地化存储
- 保护您的作答记录

### ⚡️ 秒启动

- 臃肿的医考帮 ❌
- 轻量的医不帮 ✅ 秒开无广告

### 🚀 支持所有付费题库

> 目前只有**口腔医学期末考试**的全部题库

- 登录后，可以查看题目统计信息和评论区
